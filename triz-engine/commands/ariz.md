## ABOUT ARIZ-85C

ARIZ (Algorithm of Inventive Problem Solving) is Altshuller's most sophisticated tool — a structured 9-part process for problems where direct principle application is insufficient.

## NOTE
Full ARIZ-85C implementation is being developed. For now, use `/triz:analyze` for principle-based analysis or `/triz:ifr` for Ideal Final Result formulation.

The 9 parts of ARIZ-85C:
1. Problem analysis
2. Problem model analysis
3. Formulate the IFR and physical contradiction
4. Mobilize and apply substance-field resources
5. Apply the knowledge base
6. Change or replace the problem
7. Analysis of method to remove physical contradiction
8. Application of the obtained solution
9. Analysis of steps leading to solution
